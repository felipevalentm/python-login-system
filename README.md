# Python Login System

Projeto simples de autenticação em Python desenvolvido para praticar lógica de programação e estruturas de dados.

## Funcionalidades

- Login com usuário e senha
- Validação usando dicionário
- Limite de tentativas
- Senha oculta utilizando getpass

## Tecnologias utilizadas

- Python

## Como executar

1. Clone o repositório

git clone https://github.com/felipevalentm/python-login-system

2. Entre na pasta do projeto

cd python-login-system

3. Execute o programa

python main.py

## Objetivo do projeto

Este projeto foi desenvolvido para prática de lógica de programação e conceitos iniciais de autenticação de usuários em Python.
